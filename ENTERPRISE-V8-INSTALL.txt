THE SALT ORIGIN — ENTERPRISE CMS V8
====================================

IMPORTANT
---------
This is a complete source ZIP, not a patch. Keep a backup of the current live project.
Do not copy node_modules or .next from an older project.

LOCAL INSTALL
-------------
1. Extract the ZIP into a new folder.
2. Copy .env.example to .env.local.
3. Add your real Supabase, OpenAI, Resend and WhatsApp Cloud API credentials.
4. Run:

   npm install
   npm run dev

5. Open:
   http://localhost:3000
   http://localhost:3000/admin
   http://localhost:3000/admin/blogs

CLEAN REBUILD (WINDOWS CMD)
---------------------------
   rmdir /s /q .next
   npm install
   npm run dev

CLEAN REBUILD (POWERSHELL)
--------------------------
   Remove-Item -Recurse -Force .next
   npm install
   npm run dev

SUPABASE
--------
Run the newest CMS SQL migration from the supabase folder in Supabase SQL Editor.
The existing CMS tables and architecture are preserved.

VERCEL
------
Push the extracted project to the connected Git repository. Add all environment variables in:
Vercel > Project > Settings > Environment Variables
Then deploy the main branch.
