THE SALT ORIGIN CMS V9.1 - CHANGED FILES ONLY

1. Take a backup of your current project.
2. Extract this ZIP directly inside the project root.
3. Choose Replace files in destination.
4. Run:
   npx tsc --noEmit
   npm run dev

CHANGES
- Social Media Studio now has Profile Links and Publishing Studio tabs.
- Official Facebook, Instagram, LinkedIn, TikTok, YouTube, X and Pinterest URLs can be edited, enabled and saved again.
- Existing social campaign upload, AI caption, hashtags, keywords and schedule workflow remains available.
- Blog AI Studio can manually generate today's review-ready draft.
- Daily cron now creates one researched draft per day from the saved topic focus.
- Duplicate same-day draft protection added.
- Editorial queue now includes Review and Approve & Publish actions.
- AI drafts remain Draft until manually approved.

IMPORTANT
- Auto-draft must be enabled in Blog AI Studio and Save workflow clicked.
- Vercel cron already calls /api/blog/daily-draft daily.
- OPENAI_API_KEY, SUPABASE_SERVICE_ROLE_KEY and NEXT_PUBLIC_SUPABASE_URL must be valid in Vercel and .env.local.
