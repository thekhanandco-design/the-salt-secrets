BLOG AUTOMATION
- Blog AI Studio now includes Auto-draft enabled and Approval required controls.
- Vercel Cron calls /api/blog/daily-draft every day at 06:00 UTC.
- AI drafts remain in Draft status for human review.
- Approve by changing status to Published and saving.

GOOGLE ANALYTICS
- Dashboard no longer displays invented analytics.
- Without GA4 credentials it clearly shows “Connect GA4 for real data”.
- With valid credentials it fetches real 30-day users, engagement rate and average session duration.

SOCIAL PUBLISHING
- Social Media now includes image upload, AI captions, hashtags, keywords, platform selection and scheduling.
- Scheduled records are stored in Supabase.
- Vercel Cron processes due campaigns hourly.
- Live auto-posting requires official API approval/tokens and a platform adapter for each network.
- The CMS never falsely reports a post as published when an official connection is missing.

SECURITY
- OpenAI errors are sanitized. API keys are never returned in browser alerts.
- Any API key previously exposed in a screenshot should be revoked and replaced.

VALIDATION
- TypeScript: passed with npx tsc --noEmit.
- Next.js production compilation and TypeScript stage: passed.
- Page-data collection requires working environment/database access and can take longer than the local validation timeout.
